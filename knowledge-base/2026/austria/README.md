## 🇦🇹 Key Austria Facts (2026)
1. **Tuition Fee:** Public universities cost only **€726.72 per semester** for non-EU students.
2. **Blocked Account:** None required. Bank savings statement with origin proof is sufficient.
3. **PCC & Birth Certificate:** Must be legalized via an MEA Apostille in India and translated into German.
4. **Part-time Work:** Up to 20 hours per week is legally permitted.
5. **Post-study Search:** A 12-month job search permit leading to the Red-White-Red Card.
